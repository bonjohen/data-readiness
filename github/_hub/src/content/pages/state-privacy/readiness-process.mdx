<div class="section">
  <h2>Phase Details</h2>

  <div class="phase-detail">
    <h3>1. Intake &amp; Scoping <span class="phase-duration">Week 1</span></h3>
    <p>We start by understanding your multi-state footprint and current privacy posture.</p>
    <ul>
      <li><strong>Multi-state applicability analysis</strong> &mdash; identify which state privacy laws apply based on your operations, revenue, and consumer reach</li>
      <li><strong>Current privacy program review</strong> &mdash; assess existing privacy notices, consent mechanisms, and data handling procedures</li>
      <li><strong>Consumer rights workflow assessment</strong> &mdash; evaluate current processes for responding to consumer privacy requests</li>
      <li><strong>Opt-out mechanism inventory</strong> &mdash; catalog existing opt-out mechanisms and assess GPC support readiness</li>
    </ul>
  </div>

  <div class="phase-detail">
    <h3>2. Assessment <span class="phase-duration">Week 2&ndash;3</span></h3>
    <p>We evaluate your privacy program against the requirements of all applicable state laws.</p>
    <ul>
      <li><strong>Cross-state requirement mapping</strong> &mdash; map overlapping and divergent requirements across all applicable state laws</li>
      <li><strong>Privacy notice gap analysis</strong> &mdash; identify gaps between current disclosures and multi-state requirements</li>
      <li><strong>Consent and opt-out mechanism evaluation</strong> &mdash; assess whether current mechanisms meet the strictest applicable standards</li>
      <li><strong>PIA requirement assessment</strong> &mdash; determine which processing activities trigger privacy impact assessment requirements</li>
    </ul>
  </div>

  <div class="phase-detail">
    <h3>3. Outputs <span class="phase-duration">Week 3&ndash;4</span></h3>
    <p>We deliver the artifacts that harmonize your privacy program across jurisdictions.</p>
    <ul>
      <li><strong>Harmonized compliance matrix</strong> &mdash; every requirement mapped across states, with a unified control set that satisfies all</li>
      <li><strong>Universal privacy notice template</strong> &mdash; a single privacy notice that meets or exceeds all applicable state requirements</li>
      <li><strong>Cross-state rights response procedures</strong> &mdash; standardized procedures for handling consumer requests under all applicable laws</li>
      <li><strong>Opt-out implementation plan</strong> &mdash; technical and operational plan for universal opt-out including GPC support</li>
    </ul>
  </div>

  <div class="phase-detail">
    <h3>4. Follow-on <span class="phase-duration">Ongoing</span></h3>
    <p>After the readiness sprint, ongoing monitoring keeps your program current as new laws take effect.</p>
    <ul>
      <li><strong>New state law monitoring</strong> &mdash; track newly enacted and proposed state privacy legislation for applicability</li>
      <li><strong>Privacy notice updates</strong> &mdash; update disclosures as new laws take effect or existing laws are amended</li>
      <li><strong>PIA program maintenance</strong> &mdash; maintain privacy impact assessment program as processing activities and requirements evolve</li>
    </ul>
  </div>
</div>
