AI interaction logs — the prompts consumers submit and the responses they receive — frequently contain personal information. Names, account details, health information, financial data, and other PI categories routinely appear in natural-language prompts. Under CCPA/CPRA, these logs are part of the business's PI collection and are subject to the same inventory, disclosure, access, and deletion obligations as any other PI data store.

Organizations should classify their AI interaction logs and determine which contain consumer PI, apply retention limits consistent with the data minimization principle (§1798.100(c)), and include these logs in the scope of consumer rights fulfillment. When a consumer submits a right-to-know request, the response must cover PI in prompt/response logs if those logs have been retained. When a consumer requests deletion, the logs must be purged. Failure to include AI interaction logs in the PI inventory is a common gap that becomes a compliance exposure as AI features scale.
